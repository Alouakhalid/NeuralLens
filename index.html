📖 Documentation
🎯 Features
  • ✅ Real-time Captioning — ViT + GPT-2 image-to-text generation
  • ✅ Beam Search — Trade speed vs quality (1–5 beams)
  • ✅ Temperature Control — Focused vs creative outputs
  • ✅ Incremental Training — Fine-tune without losing weights
  • ✅ Live GPU Telemetry — Real-time VRAM, util%, CPU, RAM gauges
  • ✅ API Key Auth — Secure access with expiry management
🚀 Quick Start
  1. Go to Generate tab → upload image → click Generate
  2. Go to Train tab → upload image+caption → upload sample → train
  3. Go to API Keys tab → create key → use in your app
